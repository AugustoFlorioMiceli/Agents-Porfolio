Chef Agent:

- Este Agente esta programado para servir como ayudante de una persona que que no sabe que cocinar, y por lo tanto a partir de foto, audio o texto, el agente recomendará recetas y su paso a paso para que realice el usuario.

- El agente utiliza como una de sus herramientas disponibles, la api de Tavily para poder buscar recetas interesantes en internet. Además tiene memoria ya que utiliza el metodo InMemorySaver de langgraph, por lo tanto puede recordar información que se le haya dado, pero unicamente en una misma conversación ya que no persiste en el disco (Es volátil)
